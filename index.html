HTML

  • HTML is the structure of a webpage
  • The 'main' section of a webpage stores everything visually on the webpage

CSS

  • CSS is the styling aspect of a webpage
  • You can use three different types of style sheets (In line, Style section, and external style file)
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around hte content inside an element.
  • To use a class selector, we use .class_selector to style what has been labeled with that class.

GIT

  • GIT creates a way to save and view code
  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • Git workflow:
    1. We use the 'add -A' command to commit all files in the working branch
    2. Next we need to commit changes to our local repo using 'git commit -m "message"
    3. Now we have to pull in the base branch using 'git pull origin main'
    4. Finally, pushing our new branch (as seen above using -b) to our main branch for review using 'git push origin branch name (ie: feature/add-git)

JavaScript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • Functions include 2 important factors to run: Define the function and calling the function.
  • For loops need an inital variable (x = 0); how many times to cycle through (x < list.length); and iteration (x++);
  • Triple equals returns a boolean on return (===).
